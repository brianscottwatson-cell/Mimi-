# Assets Directory

Place Mimi's profile picture here as `mimi.png` or `mimi.jpg`.

## Required Image

- **Filename**: `mimi.png` (or `mimi.jpg`)
- **Recommended size**: 256x256 pixels or larger
- **Format**: PNG (with transparency) or JPG

The image will be automatically displayed in the chat interface with:
- Neon cyan border glow
- Animated shine effect
- Rounded corners
- 48x48px display size (scaled from source)

## How to Add

1. Save Mimi's cyberpunk profile picture to this directory
2. Name it `mimi.png` or `mimi.jpg`
3. Restart the server or refresh the browser
4. The image will appear in the avatar box!

If the image fails to load, it will fallback to showing "M" as a placeholder.

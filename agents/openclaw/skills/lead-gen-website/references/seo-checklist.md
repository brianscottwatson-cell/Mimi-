# SEO Checklist — Lead Gen Websites

## Meta Tags (Every Page)
- [ ] Unique `<title>` — max 60 chars, includes primary keyword + location
- [ ] Unique `<meta description>` — max 155 chars, includes CTA
- [ ] `<link rel="canonical">` pointing to preferred URL
- [ ] Open Graph tags (og:title, og:description, og:image, og:url)
- [ ] Twitter Card meta tags

## Heading Hierarchy
- [ ] One H1 per page — includes primary keyword
- [ ] H2-H3 logical hierarchy (no skipping levels)
- [ ] Keywords in H2s where natural

## Content Quality
- [ ] Minimum 500 words for service pages, 1000 for blog
- [ ] Answers user intent (what, why, how, cost, area)
- [ ] Natural keyword integration (no stuffing)
- [ ] Mentions city/region name in body content
- [ ] Internal links between related pages (3-5 per page)

## Technical SEO
- [ ] `robots.txt` allows all key pages
- [ ] `sitemap.xml` includes all indexable pages with correct priorities
- [ ] All images have descriptive `alt` attributes
- [ ] Pages load under 3 seconds (check with Lighthouse)
- [ ] Mobile responsive (no horizontal scroll, tap targets 48px+)
- [ ] HTTPS enabled
- [ ] No broken links (404s)
- [ ] Clean URL structure (no query params for main pages)

## Structured Data
- [ ] Homepage: `LocalBusiness` schema with NAP + openingHours
- [ ] Service pages: `Service` schema with provider + areaServed
- [ ] FAQ pages: `FAQPage` schema
- [ ] Blog articles: `Article` schema with datePublished + author
- [ ] Validate at schema.org/validator or Google Rich Results Test

## Local SEO
- [ ] NAP (Name, Address, Phone) consistent across all pages
- [ ] City/region in title tags of key pages
- [ ] Service area pages for each target city (if multi-city)
- [ ] Google Business Profile linked and verified
- [ ] Local citations on key directories (Pages Jaunes, etc.)

## On-Page Conversion
- [ ] CTA visible above the fold on every page
- [ ] Phone number clickable (tel: link)
- [ ] WhatsApp link with pre-filled message
- [ ] Contact form on service pages
- [ ] Trust signals near CTAs (reviews, certifications, guarantees)
